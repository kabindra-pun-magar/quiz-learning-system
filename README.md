# 📚 Interactive Quiz-Based Learning System with Performance Analytics

A full-stack MERN application that provides an interactive quiz platform with instant feedback, performance tracking, and personalized learning insights.

## ✨ Features

### Core Features
- 🔐 **User Authentication** - JWT-based secure login/register
- 📝 **Interactive Quiz Engine** - MCQ with timer, navigation, and instant feedback
- 💡 **Instant Feedback** - Shows correct answer, user answer, and detailed explanation
- 📊 **Performance Analytics** - Track scores, accuracy, and improvement over time
- 🎯 **Weak Area Detection** - Identifies topics needing improvement
- 🎁 **Personalized Recommendations** - Suggests quizzes based on weak areas
- 🏆 **Leaderboard** - Global rankings with period filters (weekly/monthly/all-time)
- 📈 **Progress Dashboard** - Interactive charts showing learning progress
- 👑 **Admin Panel** - CRUD operations for questions and user management

### Technical Features
- 🎨 **iOS 26 Glassmorphism UI** - Modern, eye-catching design
- 📱 **Responsive Design** - Works on all devices
- ⚡ **Real-time Updates** - Instant feedback and score calculation
- 🔄 **RESTful API** - Well-structured backend architecture
- 🗄️ **MongoDB Transactions** - Data consistency
- 🛡️ **Rate Limiting** - Security protection
- 🔒 **Error Handling** - Comprehensive error management

## 🛠️ Tech Stack

### Frontend
- React 18
- Vite
- Tailwind CSS
- Recharts (for analytics graphs)
- React Router DOM
- Axios
- React Hot Toast

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT Authentication
- Bcryptjs
- Express Rate Limit
